{\centerline { {\large \textbf{ABSTRACT}}}}
~\\

Smart Attendance System using Bluetooth and Verification is designed to automate classroom attendance and reduce proxy attendance using multi-step validation. In traditional attendance systems, teachers spend significant class time taking attendance manually, and there is always a possibility of impersonation. The proposed system addresses these issues by combining session-based OTP, Bluetooth proximity verification, and face verification before recording attendance.

In the teacher workflow, a session is started for a selected class date and period. The system generates a one-time code and keeps the session active only for a limited duration. In the student workflow, attendance can be marked only after entering a valid OTP, verifying Bluetooth proximity to the teacher device, and passing face verification. The attendance entry is then stored in backend with class slot, session code, student details, and timestamp.

The application is implemented in Android Studio using Kotlin and Jetpack Compose for the user interface. Supabase is used for authentication and database operations. Kotlin Coroutines are used for asynchronous processing and live refresh features. The system also supports teacher-side live monitoring, slot-wise attendance view, and student-level reporting.

The final solution provides a practical, secure, and user-friendly attendance mechanism for classroom environments. It improves reliability of attendance records, minimizes manual effort, and lays a strong foundation for future enhancements such as analytics dashboards, anti-spoofing improvements, and advanced institutional reporting.
